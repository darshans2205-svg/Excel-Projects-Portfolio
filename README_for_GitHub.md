## 🏆 Project 1: Sales Performance Dashboard
**File:** `Project1_Sales_Dashboard.xlsx`

A complete sales analytics solution for a multi-region business with 240 rows of real-like data.

**Features:**
- 📋 **Sales Data Sheet** — 240 rows across 4 regions, 5 products, 12 months

**Excel Functions Used:**
`SUMIFS` · `VLOOKUP` · `RANK` · `IFERROR` · `IF` · `DATEDIF` · `COUNTIF`

## 🛠 How to Use VBA Macros

1. Open any `.xlsx` file in Excel
2. Press **Alt + F11** to open the VBA Editor
3. Click **Insert → Module**
4. Copy the macro code from the **"VBA Macros Code"** sheet
5. Press **F5** to run, or go to **Developer → Macros**
